笔记--移动web第三天

栅格系统

  1. 什么时候需要写栅格： 当页面的布局有行和列的布局样式就可以使用栅格 列的大小要等比例才适合
  2. 怎么使用栅格：
    1. 当前栅格需要在什么屏幕下生效 lg md sm xs
    2. 划分栅格的列数 如果等比例的就使用总列数12/当前要分的列数 = 12/3 = 4
      如果不是等比例的可以测量 或者直接估写

字体图标

  1. 原理： 通过CSS3定义一个特殊的字体（电脑上是不存在） 字体是属于你定制的 字体里面就是一堆图标
    1. 定义字体的名称(字体名称) 指定字体资源(字体文件)
    2. 使用字体 在选择器里面去使用一个字体
    3. 使用图标的编码
    4. 可以把图标封装到一个类名里面 页面中使用对应类名即可

实现商品区域的布局

  1. 使用JS插件的标签页插件实现点击切换效果
  2. 多添加几个标签页标题和面板同时修改标题和面板的id
  3. 修改标题的文字和面板里面的商品的结构
  4. 商品分为左右结构
  5. 左边里面放一个标题和商品价格
  6. 右边放一个8% 年利率
  7. 修改标题的样式 去掉边框 给选中的a加底部红色边框
  8. 修改面板商品的样式
    1. 先实现商品的左边自适应右边固定布局 左边flex:1 右边宽度写死120 左右高度都写死160
    2. 在修改左边的样式实现居中 去掉p底部间距 给左边整体添加padding
    3. 给右边设置文字大小和padding居中
    4. 给右边添加一个中间的虚线 和添加一个上下的小半圆

工具提示插件的使用

  1. 在JS插件 》 工具提示中
  2. 复制工具提示标签结构
  3. 默认工具提示没有效果的 需要手动初始化
    使用官方的JS代码去初始化
    $(function () {
    $(‘[data-toggle=”tooltip”]’).tooltip()
    })

上午复习

  1. 立即预约： 容器 左浮动右浮动 文本颜色
    修改样式的时候注意不要给容器写死高度 要写死高度尽量给子盒子写撑开父元素

  2. 商品区域： 容器 、 标签页插件 、 工具提示

    1. 标签页的插件的修改 新增一些标题和内容 id要对应上
    2. 内容结构 ：
      1.写栅格系统
      2.写商品容器
      3.商品的左右容器
      4.左边容器里面的标题和内容
      5.再写右边内容
    3. 修改样式
      1. 修改标题样式
      2. 修改内容的样式
    4. 使用工具提示插件实现提示信息： 注意插件需要初始化

移动端处理

  1. 合作伙伴超出屏幕宽度 设置flex-wrap:wrap; 换行
  2. 新闻区域内容靠左使用媒体查询屏幕宽度991以下单独处理每个li 25%宽度
  3. 商品区域的标题滚动： 使用局部滚动条 给ul外面添加一个父元素ul-wrapper 设置父元素overflow:auto;

  1. 打开bootstrap官网定制
  2. 修改一些变量或者勾选一些不需要的插件
  3. 点击底下的编译并下载

真机调试

  1. 原理： 是让手机和电脑在一个网络上使用局域网方式访问
  2. 做的事情： 就是让手机和电脑在一个网络 只能通过wifi
  3. 让手机和电脑连接同一个wifi
    1. 连接公共wifi
    2. 用手机开热点 电脑连接(推荐)
    3. 用电脑开热点 手机连接
  4. 使用手机开热点 设置 》 个人热点打开 》 电脑连接你手机开的wifi（把电脑的宽度断开）
  5. 使用电脑开启网页服务器 》 让手机访问这个服务器
    1. 打开sublime或者vscode服务器等（除了webstrome）
    2. 把localhost或者127.0.0.1换成wifi的ip地址 例如172.20.10.2
    3. 把换好后的页面刷新一下
    4. 把这个网址发给手机

安装chrome插件

  1. 点击浏览器右上角设置
  2. 打开扩展程序
  3. 开启开发者模式
  4. 把插件的crx文件拖到浏览器扩展程序界面
  5. 在右上角会出现一个图标 就是这个插件

使用chrome插件

  1. 打开页面 注意把ip换成手机热点的ip
  2. 点击左上角的插件图标
  3. 点击里面的二维码生成
  4. 扫码即可打开
